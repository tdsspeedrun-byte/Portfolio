IoT Smart Gardening System

Arduino Raspberry Pi Python SQL

A real-time data engineering pipeline designed to ingest, process, and analyze streaming environmental telemetry from automated agricultural sensors.

Project Overview

In this project, me and my teammates designed and implemented an IoT-based smart gardening system using Arduino and Raspberry Pi microcontrollers. We also used Light sensor to check if the plants are receiving adequate sunlight and Rotary sensor to adjust the brightness accordingly, along with Python for data processing and SQL for database management.

Data Pipeline Architecture

The system leverages a decoupled modern data architecture to process streaming workloads efficiently:

1. Ingestion (Arduino/Raspberry Pi)

Microcontrollers broadcast light intensity from the surrounding. When the light level falls below a certain threshold, the system alerts the user to adjust the lighting with the Rotary sensor. When it reaches the desired level, the system confirms the adjustment. It also automatically turns off the lights when not needed.

2. Processing (Python)

Python scripts consume the ingested data, perform real-time analysis, and execute control logic based on the sensor readings.

3. Storage (SQL)

Processed telemetry data is stored in a SQL database, enabling efficient querying and reporting for historical analysis.

Key Outcomes & Impact

✓ Successfully handles over 10,000 telemetry messages per minute with sub-second processing latency.

✓ Integrated automated triggering rules that trigger automated irrigation valves when soil moisture dips below a critical threshold.

✓ Produced clean data structures ready for machine learning forecasting model implementations.
